# 🧳 Lost and Found Management System

A simple and efficient web-based application designed to manage lost and found items within an organization or campus. This system helps users report lost items, submit found items, and track item status easily.

---

## 🚀 Features

* 🔐 User Authentication (Login/Register)
* 📦 Report Lost Items
* 📥 Report Found Items
* 📋 View All Items (Dashboard)
* 👤 Manage Your Reported Items
* 🔍 View Item Details
* 📸 Upload Item Images
* ✅ Claim / Confirmation System

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python
* **Database:** SQLite (local database file)

## ⚙️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/tejasdubey2468/Lost-and-found.git
cd Lost-and-found
```

2. Install dependencies (if required):

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open in browser:

```
http://localhost:5000
```
